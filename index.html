Orçamento Técnico

Realizar uma pesquisa de peças para montagem de um computador para a seguinte aplicação: programação Front-end.

O computador deverá ter os seguintes componentes:

  • Processador
  • Placa-mãe
  • Placa de vídeo
  • Memória RAM
  • Fonte ATX
  • Gabinete
  • Monitor
  • Teclado
  • Mouse
  • HD/SSD
  • Cooler + Dissipador

O orçamento disponibilizado é de R$2.500,00

Realizar pesquisa em dupla, colocar os itens nesta página organizados da forma que você achar melhor, utilizando as tags html que você já conhece.
